# TD_Dynamic_Cosim
Run co-simulation with Machine 1 and Machine 2 using remote broker on NREL HPC (or a different machine if you like).\
Machine 1 contains federates running PSS/E and OpenDSS; Machine 2 contains federate running PSCAD. 

Our paper title: Co-Simulation of PSS/E, OpenDSS, and PSCAD for Power Systems Stability Analysis With Inverter-Based Resources
https://ieeexplore.ieee.org/abstract/document/11272399
